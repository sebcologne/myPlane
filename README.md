Basic keys:

[W],[S],[A],[D] + mouselook -> MOVE
[Space] -> Jump
2x[Space] -> Fly
[E] -> INVENTORY
[F] -> Open Door/ get out (aim a cardoor/plane is a little bit tricky)

[Esc] -> options + quit
[F1] -> Emergency quit
[F2] -> Disable map loading

READ the tooltips in inventory for usage of items !
